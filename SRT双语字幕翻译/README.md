# SRT双语字幕翻译

一款简单易用的iOS应用，支持将英文字幕文件翻译成双语字幕。

## 功能特点

- 📁 支持SRT格式字幕文件上传
- 🔄 内置翻译功能，无需外部API
- 👀 双语字幕预览功能
- 💾 一键下载双语字幕文件
- 📱 专为iPhone优化设计

## 使用说明

1. **选择文件**：点击选择按钮上传SRT格式的英文字幕文件
2. **开始翻译**：点击开始翻译按钮进行字幕翻译
3. **预览字幕**：翻译完成后可预览双语字幕内容
4. **下载文件**：一键下载生成的双语字幕文件

## 技术特点

- 使用SwiftUI框架开发
- 内置简单的翻译字典
- 支持文件安全访问
- 优化的用户界面体验

## 版本信息

- 版本：1.0
- 支持系统：iOS 14.0+
- 设备：iPhone

## 隐私政策

本应用不会收集任何用户数据，所有文件处理都在设备本地完成。
