双语字幕翻译

简单易用的字幕翻译工具

应用介绍

双语字幕翻译是一款专为iOS设备设计的字幕翻译工具，支持SRT格式文件的上传、翻译和下载功能。

主要功能：

  • 上传英文SRT字幕文件
  • 自动翻译成双语字幕
  • 支持预览和下载功能
  • 完全免费使用
