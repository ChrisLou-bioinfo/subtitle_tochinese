# 🚀 快速开始：创建您的第一个iOS应用

## 📱 立即开始（5分钟）

### 步骤1：打开Xcode并创建项目

1. **打开Xcode**
   - 在Mac上找到并打开Xcode应用

2. **创建新项目**
   - 点击 "Create a new Xcode project"
   - 或者选择 File → New → Project

3. **选择模板**
   - 选择 **iOS** 标签页
   - 选择 **App** 模板
   - 点击 **Next**

4. **填写项目信息**
   ```
   Product Name: YouTubeSubtitleApp
   Team: [选择您的开发者账号]
   Organization Identifier: com.yourname
   Interface: SwiftUI
   Language: Swift
   Storage: None
   ```

5. **保存项目**
   - 选择保存位置（建议：`/Users/chris/VScode/YouTubeSubtitleApp`）
   - 点击 **Create**

## 📱 步骤3：在iPhone上运行

### 3.1 连接iPhone
1. 用USB线连接iPhone到Mac
2. 解锁iPhone
3. 在iPhone上点击"信任"这台电脑

### 3.2 选择设备
1. 在Xcode顶部工具栏，点击设备选择器
2. 选择您的iPhone（例如："Chris的iPhone"）

### 3.3 运行应用
1. 点击运行按钮（▶️）或按 Cmd+R
2. 等待编译完成
3. 首次运行会提示"无法验证开发者"

### 3.4 信任开发者证书
1. 在iPhone上打开：**设置** → **通用** → **VPN与设备管理**
2. 找到您的开发者账号
3. 点击"信任"

### 3.5 重新运行
返回Xcode，再次点击运行按钮
